Chapter 1. ACS Administrator's guide

1.1. Web Robot Detection Installation

  • install the robot-detection package using Arsdigita Package Manager (APM)

  • mount the robot-detection package under site-mapper under the root

  • set parameters for the package instance using site-mapper

  • set access permission for the pages under robot-detection using site-mapper; Disinherit permission from the root, thus making ACS-admin only person able to access it

  • build a non-password protected site starting at /robot-heaven/ (that's the default destination). If necessary to write a script to create a pseudo static HTML file appearance

  • restart AOLserver

  • visit the /robot-detection/admin to see whether your configs took effect

  • view your server error log to make sure that the filters are getting registered
